# CIS3319-6
CIS 3319 - Wireless Security - Lab 6

# Instructions to Run the Full Program
1. Clone the repository and navigate into the CIS3319-6 directory
2. Open up three terminal instances and run the following command in any 1 of them:
- 'npm install'
3. In each terminal, run 1 of the following 3 commands:
- 'npm run cert_auth'
- 'npm run server'
- 'npm run client'
4. Press the 'Enter' key in the terminal running the 'server'
5. Press the 'Enter' key in the terminal running the 'client'
6. That is all for the program.
